## 실습 2.

1. `TIL` 폴더를 만들고 git 초기화를 한다.
2. README.md를 작성한다. 
   - 나의 다짐 
     - ex. 매일 1개의 commit을 만들겠다.
     - 나는 매일 1개의 알고리즘 문제를 풀겠다.
3. 작성한 README를 commit한다.
4. 원격 저장소(github)에서 repository를 만들고 로컬(내 컴퓨터)과 연결한다.
   - hint: add remote
5. 최종 push하여 원격 저장소에 업로드한다.
